Free Town Guide Generator

Introduction

The Free Town Guide Generator will generate simple PDF posters which are guides to a particular area. The poster contains a map, and a selectable set of features (a street index, or lists of pubs, restaurants, shops, banks etc.). The Data is derived from the project, which is a freely available map dataset, that can be edited by anyone. Generate a Town Guide

To generate a town guide you need to know the latitude and longitude of the bottom left corner of the area you are interested in. You also specify the size of the area to be mapped in 1km units. You can try the generator here. Please note though that it is running on a very small computer, and can only deal with one request at a time (I do not know what will happen if more than one person tries to use it at once!). I will add a queue to make it multi-user friendly one evening next week. The source code for the townguide generator is at http://code.google.com/p/townguide - I will move it to a separate project next week.
